# echo
Turn console.log into echo. That simple

### Importing  
```javascript
const echo = require('./index').echo;
```  
  
### Parameters  
(valor)

| PARAMETER  | VALUE           | OBS                                           |
|------------|-----------------|-----------------------------------------------|
| value      | Any value       | Any value you want displayed on the console.  |
  

### Usage example 

```javascript
const echo = require('./index').echo;
echo('Hello world!!!');
```
### Return  

Returns the value indicated in the parameter.

> Hello world!!!
